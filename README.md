# GPA Calculator - Java

A simple Java console application that calculates a student's GPA 
based on course credit hours and letter grades.

## Features
- Enter number of courses
- Enter credit hours for each course
- Enter letter grade (A, B+, C, etc.)
- Automatically calculates GPA

## Technologies Used
- Java
- OOP Concepts
- Loops
- Conditional Statements

## How to Run
1. Compile the file:
   javac GPACalculator.java
2. Run:
   java GPACalculator
